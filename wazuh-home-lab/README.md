# Wazuh SIEM Home Lab

## Overview
This project documents the setup and investigation of a home Security Information and Event Management (SIEM) lab using Wazuh.

The lab was created to simulate real SOC monitoring environments and practice threat detection and incident investigation.

## Lab Environment

Components used:

- Wazuh SIEM
- Kali Linux (attacker machine)
- Windows VM (target machine)
- Log monitoring and alert analysis

## Objective

The goal of this lab was to:

- Monitor security events
- Generate attack activity
- Detect suspicious behaviour
- Investigate alerts in the SIEM dashboard

## Example Attack Scenarios

Activities performed in the lab include:

- Port scanning
- Brute force login attempts
- Suspicious network activity
- Log analysis of authentication events

## Skills Practiced

- SIEM monitoring
- Security event investigation
- Log analysis
- Threat detection
- Incident response workflow
